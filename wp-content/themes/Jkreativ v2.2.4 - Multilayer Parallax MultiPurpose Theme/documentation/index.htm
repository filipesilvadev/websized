Wordpress Installation

This is short walkthrough of Wordpress installation. This part is usefull if you install Wordpress theme for the first time. Before start, you must have a server that support PHP and enable access to MySQL databases.

Download Wordpress Download latest zip format wordpress from http://wordpress.org/download/.

After download wordpress, extract the file. And you will get folder named wordpress. Upload all file from those folder to your hosting server. You will need FTP credential and FTP Client for uploading wordpress. Contact your hosting for your site FTP credential and If you don't have ftp client, you can download it here.

Wordpress Requirement Before you buy any hosting server, you need to make sure that your hosting able to host wordpress. We recommended you to check PHP & MySQL version where you host your website. The latest version they have will be better..

After finish uploading wordpress, now try to access yor website URL from your browser. And you should see wordpress complaining cannot find wp-config.php. Click Create a Configuration File, and you will prompt that you need to know Database Name, Database Username, Database Password, Database Host, and Table Prefix. Click Let's Go to fill all detail of your wordpress installation.

And after all setup, you need to fill your site information (Site title, username, password, etc). But all of option can be changed latter. Bellow you can find image recap for wordpress installation

Login to admin page

After you finish installation, you can login to admin page with username and password that you enter before. Point your browser url to http://yourdomain.com/wp-admin/

This is only short guide for installing wordpress. You can find more complete installation guide from wordpress itself by clicking this link

Themes Installation & Activation

Installation

Now you able to login into wordpress admin page, next step is installing themes. Installing JKreativ Themes is quite easy. Basically install this themes is almost the same as installing other themes. This part will guide you to how install JKreativ Themes.

After downloading JKreativ package from Themeforest, you will find zip file with name like 'JKreativ_vxxx.zip'. This file is main JKreativ Themes. There are two way to install this theme :

  1. Install themes via Wordpress themes uploader

    To install themes via wordpress uploader, you will need zip file from JKreativ name like 'JKreativ_vxxx.zip'.

    Go To : Appearance » Themes » Click on Install Themes tab » click on Upload link

    Click Browse, then locate theme file with zip format. Click open, then click install now.

  2. Install themes by uploading file (via FTP)

    You will need FTP credential and FTP Client for uploading theme. Contact your hosting for your site FTP credential and If you don't have ftp client, you can download it here.

    Info : Extract JKreativ_vxxx.zip, you will get folder named JKreativ_vxxx, upload this folder to Wordpress Instalation folder » wp-content » themes

    Warning : You should upload your folder in right directory, and check again after upload file (many case undetected themes because themes extracted wrapped again with its folder like "themes/JKreativ/JKreativ"

Plugin - Mega Menu

Jkreativ theme comes with mega menu feature that implemented custom version of "Mega Main Menu" plugin.

1. Get Started

  1. Make sure Mega Main Menu plugin is installed & actived
  2. Go to Appeareance > Mega Main Menu to activate mega menu :
    • side_navigation : enable mega menu on side navigation / vertical navigation
    • top_navigation : enable mega menu on top navigation / horizontal navigation
  3. Each menu location has General Option: Layout Options so you can customize it individualy. Here are full options that you can customize:
    • Location Of Icon In First Level Elements : Choose where to locate icon for first level items.
    • Separator : Select type of separator between the first level items of this menu.
    • Rounded Corners : Select the value of corners radius.
    • Dropdowns Animation : Select the type of animation to displaying dropdowns. Warning: Animation correctly works only in the latest versions of progressive browsers.

If you don't see any menu that you can activated, please go to : Appearance > Menu and create menu. Also please assign your menu on Side navigation or Top navigation, so you can assign it on mega menu

 

2. Setup Mega Menu

After activate mega menu on Mega Main Menu option, next step is setup menu structure from Menus, go to: Appeareance > Menus

Explanation

  1. Icon of This item : set this if you wish to add icon
  2. Hide Icon of This item : check this if you want to hide icon
  3. Hide Text of This item : check this if you want to hide text
  4. Disable Link : check this if you want to disable link
  5. Submenu Type : select Multicolumn Dropdown to make menu have multi column child (mega menu)
  6. Side of dropdown elements : behavior of dropdown
  7. Submenu Columns : number of columns of mega menu
  8. Additional Right Padding : add space at right side of menu
  9. Additional Bottom Padding : add space at bottom side of menu
  10. Dropdown Background Image : add image for menu background

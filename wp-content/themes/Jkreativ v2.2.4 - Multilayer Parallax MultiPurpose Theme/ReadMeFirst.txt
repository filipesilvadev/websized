Welcome To Jkreativ
 - You will need to upload jkreativ-themes.zip as your themes.
 - Please install necessary plugin first
 - Dummy Content available for one click install. Go to WordPress admin panel > Jkreativ > Import Dummy Content
 - for more information, please read documentation, you can also find documentation online at : http://jkreativ.jegtheme.com/documentation for more updated documentation
 - if you find any problem during installation or setting up themes, please go to http://support.jegtheme.com
 - If you going to make heavy change on themes, you can use jkreativ child. but if you going to make simple css change, please use additional css feature on Jkreativ Dashboard
